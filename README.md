# Airbnb Clone

A full-stack Airbnb-inspired web application built using the MERN stack.

## Features

* Property Listings
* Search by Location
* Property Ratings
* Favorite Button
* REST API Integration
* MongoDB Database
* Responsive UI

## Tech Stack

### Frontend

* React.js
* Axios
* CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

## API Endpoint

GET /api/properties

Returns all available properties.
